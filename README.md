# Advanced-Pattern-Problems

A collection of advanced pattern-based problems in Java, focused on mastering nested loops, coordinate management, and symmetrical logic. This repository documents the step-by-step implementation of complex shapes to strengthen logical thinking and iterative control.

## 🚀 Goal
To master complex nested loop structures and spatial logic in Java, moving from basic patterns to advanced symmetrical designs.

## 📂 Patterns Implemented
- [x] **Butterfly Pattern**: Logic for symmetrical upper and lower halves with space management.
- [x] **Solid Rhombus**: Implementation of slanted solid shapes.
- [x] **Number Pyramid**: Rows containing sequential increments.
- [x] **Palindromic Pyramid**: Advanced numerical patterns with descending and ascending sequences.
- [x] **Diamond Pattern**: Combined upper and lower triangular logic.

## 🛠 Concepts Covered
- **Nested Loops**: Managing the interplay between outer (i) and inner (j) loops.
- **Coordinate Management**: Manipulating print statements to achieve specific visual outputs.
- **Symmetrical Logic**: Using loop variations to create mirror-image patterns.
